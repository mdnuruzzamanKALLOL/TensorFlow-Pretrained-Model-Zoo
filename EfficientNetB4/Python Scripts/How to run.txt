EfficientNetB4 — Python Scripts
===============================

1. ARCHITECTURE ONLY
   python EfficientNetB4.py

2. TRAINING FROM SCRATCH
   Prepare your dataset:
     dataset/
       train/  class_a/  class_b/ ...
       val/    class_a/  class_b/ ...

   python train.py

   Outputs:
     efficientnet_b4_best.keras    (best val_accuracy checkpoint)
     efficientnet_b4_final.keras   (final epoch weights)

3. INFERENCE
   Edit CLASS_NAMES in inference.py, then:
   python inference.py path/to/image.jpg

Note: Input images are resized to 380x380 internally.
