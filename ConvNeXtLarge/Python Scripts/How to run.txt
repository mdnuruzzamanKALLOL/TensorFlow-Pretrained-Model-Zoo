ConvNeXtLarge — Python Scripts
==============================

1. ARCHITECTURE ONLY
   python ConvNeXtLarge.py

2. TRAINING FROM SCRATCH
   Prepare your dataset:
     dataset/
       train/  class_a/  class_b/ ...
       val/    class_a/  class_b/ ...

   python train.py

   Outputs:
     convnext_large_best.keras    (best val_accuracy checkpoint)
     convnext_large_final.keras   (final epoch weights)

3. INFERENCE
   Edit CLASS_NAMES in inference.py, then:
   python inference.py path/to/image.jpg

Note: When loading a saved model, pass
   custom_objects={"LayerScale": LayerScale}
   because LayerScale is a custom Keras layer.
