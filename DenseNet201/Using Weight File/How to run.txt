DenseNet201 — Using Weight File
===============================

Prerequisites
-------------
pip install tensorflow scikit-learn matplotlib

1. LOAD PRETRAINED  (ImageNet weights, predict only)
   python load_pretrained.py path/to/image.jpg

2. FEATURE EXTRACTION  (Phase 1: frozen base, new head)
   Prepare dataset/train and dataset/val directories.
   python feature_extraction.py
   Output: densenet201_fe_best.keras

3. FINE-TUNING  (Phase 1 + Phase 2: progressive unfreeze)
   Prepare dataset/train and dataset/val directories.
   python fine_tuning.py
   Output: densenet201_fine_tuned_best.keras

Notes
-----
- DenseNet preprocessing: preprocess_input from
  tensorflow.keras.applications.densenet
  (divides by 255, subtracts ImageNet channel mean, divides by std).
- Phase 2 unfreezes the last 40% of base layers at lr=1e-5.
- Saved models use .keras format (TF SavedModel-compatible).
